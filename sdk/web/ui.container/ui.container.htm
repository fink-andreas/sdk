ui.container

function Container(left, top, width, height, parent, content)

Creates an empty container (div) element.

[left], [top]
If one of these arguments are provided, the div is created with absolute position on these coordinates. If these argumends are undefined, the element position is relative.
[width], [height]
If provided the div size is set accordingly. A value of -1 sets the width/height to 100%.
[parent]
If parent is set, the container element is automatically appended to the child nodes of the parent.
[content]
If content is given, it is automatically appended to the child nodes of the container.

Members

container
The container element of the container class.

Functions

setContent (content)
Empties the current container nodes and appends the content to the container node.
show ()
Shows the content again, if the function hide was called.
hide ()
Hides the content.
setBorder (borderWidth)
Sets a border of the container, default border width is 1px.
setBorderWidth (left, top, right, buttom)
Sets the four widths of the border.
addElement (element)
Adds an element to the container.
setOnClick (onclick, obj)
Sets the onclick handler. The onclick handler is a function with the argument obj.
setOnMouseEnter (funcOnMouseEnter, obj)
funcOnMouseEnter is called with obj on the onmouseenter event on the container node.
setOnMouseLeave (funcOnMouseLeave, obj)
funcOnMouseLeave is called with obj on the onmouseleave event on the container node.
