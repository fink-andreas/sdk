ui.hider

innovaphone.ui.Hider(container, caption, funcClickOnCaption)

An element with a caption and a hider button to hide/show the element.

container
content
returns the content element
hiddenWidth
returns the width of the hidden element
setOnShow(funcOnShow)
sets a function which is called if the element is shown
setOnHide(funcOnHide)
sets a function which is called if the element is hidden
setCaption(caption, [funcClickOnCaption])
sets a new caption and an optional onClick function for the header
