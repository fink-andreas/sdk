ui.anchor

innovaphone.ui.Anchor(container, url, text, width, height, left, top, parent)

Creates an anchor field.

Arguments

container
url
the url
[text]
default the url
[width]
field width, default 200
[height]
field height, default 20
[left, top]
If set, the element is positioned absolute
[parent]
If set, the element is appended as child

Functions

container
the container (span) which contains the anchor
getUrl()
returns the url
setUrl(url)
to set the url
setTitle(title)
to set the title, default the url
setTarget(target)
to set the target, default '_blank'
value()
returns the value, default the url
setOnClick(onClick)
sets a function which is called when the link is clicked
