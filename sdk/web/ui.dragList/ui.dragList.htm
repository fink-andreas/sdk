ui.dragList

innovaphone.ui.dragList(container

Creates a display list that can be edited by the user by dragging items and deleting them

Arguments

container
container for list

Functions

AddItem(content)
Add a new string item to the list. The string is specified by content
GetElements
Return an array with the current string items in the list in the order specified
setOnRemoveElement
Add a event fired when a item is removed from the list
