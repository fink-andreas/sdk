ui.confirmationpopup

innovaphone.ui.ConfirmationPopup(container, textQuestion, textYes, textNo, funcYes, funcNo)

Opens a popup with Yes/No buttons and a question text.

  • container
  • textQuestion
  • textYes: text of the yes button
  • textNo: text of the no button
  • funcYes: function which is executed on Yes click. If funcYes returns true, the popup closes automatically
  • [funcNo]: optional function which is executed on No click
close()
closes the popup
