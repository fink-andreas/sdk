ui.drag

innovaphone.ui.Drag

You can't create any instance of Drag(). When the javascript file has been loaded, '''innovaphone.ui.Drag''' is an instance of the class '''Drag'''. The class exposes just one function:

makeDraggable(element, grabber)
after calling this function the element is draggable with the grabber
element
the element, which is dragged
grabber
the grabber element which is used to drag the above element
