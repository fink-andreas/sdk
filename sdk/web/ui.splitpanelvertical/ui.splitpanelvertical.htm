ui.splitpanelvertical

innovaphone.ui.SplitPanelVertical(container, storageId, height)

Creates a vertical splitpanel. Currently only basic implementation with fixed top/down division.

Arguments

container
the container the splitpanel is added to
[storageId]
currently unused.
[height]
currently unused. Fix division: top 46%, down 54%

Functions

getTop()
returns the top div
getDown()
returns the down div
cleanupTop()
removes all top elements
cleanupDown()
removes all down elements
