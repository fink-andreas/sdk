ui.splitpanel

innovaphone.ui.SplitPanel(container, storageId, width)

Creates a spit panel where a splitter can be moved to resize the left/right side of the content.

Functions

getLeft()
returns the left element node
getRight()
returns the right element node
getLastWidth()
returns the last width of the splitpanels left side
redraw(width)
redraws the splitpanel with the fix left width
innerSplitPanel()
to define the splitpanel as an inner splitpanel, which left side is same colored as the right one
cleanupLeft()
removes all nodes from the left side
cleanupRight()
removes all nodes from the right side
