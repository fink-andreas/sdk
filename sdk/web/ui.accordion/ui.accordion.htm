ui.accordion

innovaphone.ui.Accordion(container, left, top, accWidth, accHeight)

Creates an Accordion display

Arguments

container
container for the accordion
left
left position relative to the container
top
top position relative to the container
accWidth
the maximum width of the container
accHeight
the maximum height of the container

Functions

addSection(name, container)
Add a new accordion section with the title "name" and content from container
removeSection(name)
Remove the section with the title "name" from the accordion
setOnChangeSection(onChange)
Add listener for handling the changing the accordion section
getExpandedSection
Returns the name of the expanded section
getExpandedMaxHeight
Returns the maximum height of the expanded section body
