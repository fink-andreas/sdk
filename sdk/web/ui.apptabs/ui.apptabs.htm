ui.apptabs

innovaphone.ui.AppTabs(container, storageId, obj)

Adds tabs to the left and/or right of the page.

container
[storageId]
if given, stores the active tab inside the localstorage
[obj]
Object stored in the tab. Passed as argument in oninit
addApp(name, onInit)
adds an application tab to the left
addRight(element)
adds an element to the right
show()
shows the last active (localStorage) tab
focus(text)
Shows the Tab identified by text
