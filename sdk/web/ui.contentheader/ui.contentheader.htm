ui.contentheader

function ContentHeader(container, title)

creates a ContentHeader element with a title

[container]
optional container element
title
the header title text

Members

container
The container element.
content
The content element which contains the content below the header.

Functions

appendContent (element)
Appends the element node to the content node.

Removes all child nodes from the content node.
setTitle (text)
