ui.editpopup

innovaphone.ui.EditPopup(container, headText, columnCount, width, height, textOk, textCancel, funcOk, funcCancel)

Opens a popup with Ok/Cancel buttons. The popup can be used like an innovaphone.ui.Popup.

  • ... (see innovaphone.ui.Popup)
  • textOk: text on the ok button
  • textCancel: text on the cancel button
  • funcOk: function which is called on Ok. If this function returns true, the popup is closed automatically
  • [funcCancel]: function which is called on Cancel.
close()
closes the popup
addElement(element, fillLine)
see innovaphone.ui.ElementsContainer
adjustColumnWidth(column)
see innovaphone.ui.ElementsContainer
setOnClose(funcOnClose)
function is called when popup is closed
