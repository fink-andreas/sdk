ribbon

innovaphone.ui.Ribbon(container)

Creates a ribbon area, which can be filled with ribbon groups from the left and elements can be added to the right side.

addGroup(text)
creates, appends and returns class Group with footer '''text''' to the left elements
addRight(element)
appends an existing element to the right side
getContent()
returns the content element

class Group

A ribbon group which stores the buttons.

container
addButton(text, iconUrl, iconPosX, iconPosY, funcOnClick)
adds and returns the class Button
text
caption of the button
iconUrl
the URL to the icon (see example ui.ribbon.example.png)
The icon must be a sprite icon with icons in the size 25x25px.
There must be four icons next together on the x axis for each icon symbol:
  • Black icon at 0px
  • White icon at 25px
  • Blue icon at 50px (CSS hover)
  • Orange icon at 75px (CSS active)
The image states are automatically controlled by the CSS class ijs-sprite-25px
[iconPosX]
the X position of the icon inside the image
[iconPosY]
the Y position of the icon inside the image
[funcOnClick]
an optional onclick function for the button
disable()
disables all buttons of the group
enable()
enables all buttons of the Group
addElementButton(element, text)
to add and return an element button containing the predefined element
element
the predefined element, e.g. an input field
text
caption of the element button

class Button

A ribbon button.

setOnClick(funcOnClick)
funcOnClick can be null
disable()
disables the button
enable()
enables the button

class ElementButton

A ribbon element button.

setOnClick(funcOnClick)
onClick function (optional)
disable()
disables the element button
enable()
enables the element button
