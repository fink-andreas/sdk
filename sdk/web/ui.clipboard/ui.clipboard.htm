ui.clipboard

innovaphone.ui.ClipBoard(container, value, width, height, bordered)

contains a textarea whose text is copied to the clipboard. If bordered the textarea is disabled, otherwise the textarea is readonly.

Arguments

container
[value]
initial text
[width]
textarea width, default 200
[height]
textarea height, default 20
[bordered]
boolean to get a bordered textarea

Functions

container
the container (span) which contains the textarea
value()
return the text in the textarea
setValue(value)
to set the text in the textarea
setWidth(width)
to set the textarea width
setHeight(height)
to set the textarea heigth
copyTextToClipBoard(value)
copies the text in the textarea into the clipboard. If value is defined the text in the textarea is set to value, then the textarea text is copied into clipboard
