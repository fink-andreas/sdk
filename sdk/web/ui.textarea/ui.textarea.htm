ui.textarea

innovaphone.ui.TextArea(container, value, width, height, left, top, parent)

Creates a textarea.

Arguments

container
[value]
the initial textual value
[width]
field width
[height]
field height
[left, top]
If set, the element is positioned absolute
[parent]
If set, the element is appended as child

Functions

container
the container (span) which contains the textarea
value()
returns the textual value
setValue(value)
to set the textual value
setWidth(width)
to set the width
setHeight(height)
to set the height
setTitle(title)
to set the title
disableResize()
to disable resize
