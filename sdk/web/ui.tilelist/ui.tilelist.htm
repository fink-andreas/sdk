ui.Tilelist

innovaphone.ui.Tilelist(container, tileWidth, defaultBkColor, defaultTextColor, width, height)

Creates a display list using tiles. Background and text color of the tiles can be configured for the whole list and for each element

Arguments

container
container for list
tileWidth
the width of a tile in pixels -> REQUIRED
defaultBkColor
the background color of the tile for the whole list -> REQUIRED
defaultTextColor
the background color of the text in the tile for the whole list -> REQUIRED
width
width of the container
height
heighth of the container

Functions

AddItem(content)
Add a new string item to the list. The string is specified by content
AddSubtextToItem(itemText, subText)
Add a sub text on a new line for the specified item. The item is specified by itemText
AddSubtextToElement(element, subText)
Add a sub text on a new line for the specified item.
RemoveElement(text)
Remove the element specified by text
RemoveAllElements()
Remove all elements
onClick
Set the onClick function. The function will take as parameter the container of the clicked element
GetElements
Return an array with the current string items in the list in the order specified
ChangeElementColors(elementText, backgroundColor, textColor)
Change the colors for a specified item. The item is specified by elementText. If a color is not specified, the default one specified in the constructor is used
ChangeColorsToElement(element, backgroundColor, textColor)
Change the colors for a specified element. If a color is not specified, the default one specified in the constructor is used
