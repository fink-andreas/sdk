ui.listview

innovaphone.ui.ListView(container, storageId, funcItemSelectionChanged, multiSelect, noHeader)

A sortable and resizable list view.

  • [container]
  • [storageId]
    remembers the width of a columns
  • [funcItemSelectionChanged]
    if this function is given, items can be selected. On item selection, this function is called (without parameters)
  • [multiSelect]
    multiple items can be selected
  • [noHeader]
    the list view won't have a column header

Call resize after you added all columns/rows to resize the scrollbar.
