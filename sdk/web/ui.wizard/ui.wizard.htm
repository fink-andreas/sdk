ui.wizard

innovaphone.ui.Wizard(container, headText, columnCount, width, height, funcOnFinish, textNext, textPrevious, textFinish)

The wizard has several pages. A page is an instance of '''innovaphone.ui.Popup'''.

  • container - height: see innovaphone.ui.Popup
  • funcOnFinish: called if finish button is pressed. If funcOnFinish returns '''true''', the wizard is implicitly closed.
  • textNext: text shown on next button
  • textPrevious: text shown on previous button
  • textFinish: text shown on finish button
addPage(pageHeadText, lastPage, funcOnNext)
adds a new page and returns an instance of innovaphone.ui.Popup
pageHeadText
an additional header text for the current wizard page
[lastPage]
boolean, set true for the last page
[funcOnNext]
a function which will be called on use of a next button. If function is set and it returns true, the next page is shown
showPage(pageNumber)
shows the page pageNumber
close()
closes the wizard
