ui.icon

innovaphone.ui.Icon(container, iconUrl, iconPosX, iconPosY, width, height, left, top)

Creates an Icon

container
iconUrl
the URL of the icon
[iconPosX]
if the icon is inside a sprite image, this defines the x position inside this image
[iconPosY]
if the icon is inside a sprite image, this defines the y position inside this image
[width]
default 20px
[height]
default 20px
[left]/[top]
if set the element is positioned absolute at this postion

Members

container
the container element
icon
the icon element

Functions

setWidth (width)
sets the width of the icon element
setHeight (height)
sets the height of the icon element
addSpriteClass (className)
adds the class to the icon element there are some default sprite classes inside innovaphone.styles.css, e.g. ijs-sprite-25px
the hover image is 25px further on the x axis, the active image 25px further
setTitle (title)
sets the title aka tooltip of the icon
show ()
shows the icon again, if the function hide was called
hide ()
hides the icon
setOnClick (clickFnc, obj)
sets a function which is executed with the obj parameter on click
setOnMouseDown (mouseDownFnc, obj)
sets a function which is executed with the obj parameter on mouse down
setOnMouseUp (mouseUpFnc, obj)
sets a function which is executed with the obj parameter on mouse up
setOnMouseEnter (mouseEnterFnc, obj)
sets a function which is executed with the obj parameter on mouse enter
setOnMouseLeave (mouseLeaveFnc, obj)
sets a function which is executed with the obj parameter on mouse leave
