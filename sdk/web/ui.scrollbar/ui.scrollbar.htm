ui.scrollbar

innovaphone.ui.ScrollBar(container)

getContent()
returns the content node of the scrollbar
appendContent(element)
appends the element to the content node
removeContent(element)
removes the element from the content node
clearContent()
removes all elements from the content node
insertContentBefore(element, refelement)
insert element before refelement
resize()
resizes the scrollbar to fit to the content
scrollToBottom()
scrolls to the bottom
scrollIntoView(element)
scrolls until the element is inside the view
setWidth(width)
sets the width of the scrollbar content. This might be neccessary if the content does not have a fixed width.
setMaxHeight(height)
sets the maximum height of the scrollbar content. This might be neccessary if you can't determine the height.
setOnScroll(function)
Set a onScroll event handler. The onScroll handler is called with the scrollTop value as argument
