button

innovaphone.ui.Button(container, text, funcOnClick, left, top, obj)

Creates a button with an onClick function.

Arguments:

container
HTML element into which the button is placed. If undefined a span element is created#
text
Text of the button
[funcOnClick(element)]
On click handler. The button object is passed as argument
[left], [top]
If set, the button is positioned absolute at this postion
[obj]
An optional javascript object, which is attached to the button object as member 'obj'. This can be used to store context information, to be used in the onclick handler

Public functions:

setText(text)
sets the text of the button
click()
performs a click on the button
getButtonElement()
return the DOM button element
disable()
disables the button
enable()
enables the button
