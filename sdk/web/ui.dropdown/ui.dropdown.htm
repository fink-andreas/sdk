ui.dropdown

innovaphone.ui.DropDown(container, width, invertedColors, userInput, maxInputLength, left, top, parent)

Creates a DropDown box.

Arguments

width
width of the dropdown box, e.g. '''135'''
[invertedColors ](boolean)
can be used either for a dark or light background
[userInput] (boolean)
if set, the user can input an own value
[maxInputLength] (int)
[left, top]
If set, the element is positioned absolute
[parent]
If set, the element is appended as child

Keyboard Handling

  • ArrowDown: if not opened, opens the DropDown box, otherwise jumps to the next item
  • ArrowUp: if not opened, opens the DropDown box, otherwise jumps to the previous item
  • Escape: closes the dropdown box
  • Enter: if the dropdown box has an editable input element, the dropdown box is closed with the new input text except that an item is selected with ArrowUp/ArrowDown
  • Character: without an editable input field, searches and jumps to an item which starts with this character (or string if multiple characters are entered).
    Entered text is cleared after one second.
