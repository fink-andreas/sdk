PBX Websocket APIs

There are several ways how Websocket connections can be established from or to the innovaphone PBX. Regardless how the connection is established APIs provided by the PBX can be used on the connection, as long as by some administrativ configuration the connection is authorized for the use of the API.

The APIs are addressed by the "api" property. The following APIs are implemented.

PbxAdminApi
An API which allows to read and write the configuration of the PBX.
