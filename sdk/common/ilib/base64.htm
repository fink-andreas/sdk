base64

dword decode_base64(const char * in, byte * out, int n)

Decodes a base64 string to binary data.

const char * in
Character buffer containing the BASE64 encdoed data.
byte * out
The buffer where the decoded data should be written to. The application must make sure that the buffer is big enough to hold the decoded data.
int n
The number of bytes of the BASE64 string in the in buffer.
return value
The number of bytes written to the out buffer.

dword encode_base64(const byte * in, char * out, int len)

Decodes a base64 string to binary data.

const byte * in
A buffer containing the raw data to be encoded.
char * out
The buffer where the encoded BASE64 string should be written to. The application must make sure that the buffer is big enough to hold the BASE64 string and a terminating NULL byte.
int len
The number of bytes in the in buffer to be encoded.
return value
The number of bytes written to the out buffer.
