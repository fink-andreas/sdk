asn1

This is part of the ASN1 encoder/decoder library. The class defined in this file are allocated statically to create the definition of the ASN1 data structures.

The asn1_per or asn1_ber are used together with these definitions to do the actual Basic Encoding Rule (asn1_ber) or Packet Encoding Rules (asn1_per) encoding and decoding.

asn1_out

Definition of virtual class to write encoded data. Used by the encoder.

asn1_in

Definition of virtual class to read encoded data. Used by the decoder.

asn1_context, asn1_tag

class to holding the actual decoded data or holding the data to be encoded

Supported data types

For each supported datatype a class is available for the definition, which uses the class asn1 as base class.
