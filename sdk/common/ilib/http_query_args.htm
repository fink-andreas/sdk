http_query_args

A class to parse HTTP query arguments, e.g. ?arg1=value1&arg2=value2

class HttpQueryArgs(char * query)
note that the '''query''' will be '''modified'''!
Current()
returns HttpQueryArg of current argument or NULL
Next()
returns HttpQueryArg of next argument or NULL
class HttpQueryArg()
name
the name of the argument
value
the already URL decoded value of the argument 
char * query = "?arg1=value1&arg2=value2";
for (HttpQueryArgs args(query); args.Current(); args.Next()) {
    HttpQueryArg * arg = args.Current();
    debug->printf("%s=%s", arg->name, arg->value);
}
