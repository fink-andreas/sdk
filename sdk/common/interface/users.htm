class IUsers

static users_password_change_error_t ChangePassword(const char * user, const char * oldPassword, const char * newPassword)
Changes the password of a system user. The user name, the old and new password must be given. Returns USERS_PASSWORD_CHANGE_SUCCESS on success.

Code Example 

users_password_change_error_t error = IUsers::ChangePassword("root", "iplinux", "superSecret");
